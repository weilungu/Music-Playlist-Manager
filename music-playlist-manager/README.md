# 音樂播放清單管理器

這是一個音樂播放清單管理器專案，旨在提供用戶一個方便的方式來管理他們的音樂播放清單。該專案使用多種資料結構來實現不同的功能，包括鏈結串列、佇列、雜湊表和二元搜尋樹。

## 專案功能

- **新增/刪除歌曲**：用戶可以輕鬆地將歌曲新增到播放清單中或從中刪除。
- **隨機播放**：支持隨機播放歌曲的功能，讓用戶享受不同的音樂體驗。
- **搜尋歌曲**：用戶可以根據曲名或歌手快速搜尋歌曲。
- **播放列表前後切換**：用戶可以在播放列表中前後切換歌曲，方便控制播放順序。

## 資料結構

- **鏈結串列**：用於管理播放清單，提供新增、刪除和遍歷歌曲的方法。
- **佇列**：用於管理即將播放的歌曲，支持入隊和出隊操作。
- **雜湊表**：用於快速查詢歌曲，根據曲名或歌手進行搜尋。
- **二元搜尋樹**：用於排序所有歌曲，提供插入、刪除和查詢的方法。

## 使用方法

1. 克隆此專案到本地。
2. 使用瀏覽器打開 `src/index.html` 以查看應用程式介面。
3. 根據需要新增或刪除歌曲，並使用搜尋功能查找特定歌曲。
4. 享受隨機播放和播放列表的切換功能。
